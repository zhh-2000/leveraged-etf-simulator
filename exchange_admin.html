🏦 配资业务总览

作为交易所/配资方，我们为客户提供杠杆资金。每个杠杆ETF产品的运作机制如下： 客户自有资金 + 交易所配资资金 → 买入标的资产 → 每日重新平衡杠杆比例。 交易所收取配资利息作为收入，同时承担客户亏损超过保证金的风险敞口。

🏛 杠杆ETF = 交易所为客户配资

杠杆ETF的本质就是交易所/做市商为客户提供杠杆资金进行配资交易。让我们来看一个实际的例子：

📝 案例：客户买入3倍茅台ETF

客户自有资金 ¥1,000,000
交易所配资 (2倍杠杆) + ¥2,000,000
总买入金额 ¥3,000,000
实际杠杆倍数 3倍

🔄 每日重置 = 每日重新配资

每日重置的实质是：每天收盘后，交易所根据客户最新的资产状况，重新计算配资金额。

📊 重置计算过程

场景 标的涨+10% 标的跌-10%
客户资产变化 +30% → ¥1,300,000 -30% → ¥700,000
交易所调整配资 需追加配资至 ¥2,600,000 需减少配资至 ¥1,400,000
总资产（维持3倍） ¥3,900,000 ¥2,100,000

⚖️ 交易所的风控策略

🛡 保证金制度

• 初始保证金: 客户自有资金的100%作为保证金
• 维持保证金: 保证金比例需维持在70%以上
• 预警线: 保证金比例低于50%时触发预警
• 强平线: 保证金比例低于30%时强制平仓

📊 集中度限制

• 单一品种配资不超过总资金池的40%
• 单一客户配资不超过总资金的20%
• 高波动品种(3倍)配资系数额外打8折
• 每日收盘后重新评估集中度风险

📈 波动率风控

• 实时监控标的波动率变化
• 波动率突升时自动调低杠杆上限
• 市场波动率 > 30% 时限制3倍产品开仓
• 重大事件时启动熔断保护

📋 强平机制

• 触发强平线后立即发出强平通知
• 优先平掉亏损最大的持仓
• 强平后归还交易所配资本金
• 剩余资金退还给客户

💰 交易所盈利模型

收入来源 配资利息收入 按配资金额 × 年化利率 / 365 每日收取
成本 资金成本 + 运营成本 交易所自有资金的机会成本或融资成本
主要风险 客户穿仓风险 极端行情下客户亏损超过保证金，交易所需承担损失
盈利关键 利差 + 风控 收取的利息 > 资金成本 + 坏账损失，且风控到位避免穿仓
