# 杠杆ETF模拟交易平台 ⚡

一个基于中国A股标的（茅台、兆易创新、宁德时代）的杠杆ETF模拟交易系统，包含**交易端**和**交易所管理后台**，帮助理解杠杆ETF的每日重置机制、波动损耗等核心概念。

## 功能模块

### 🎯 交易端 (`leveraged_etf_simulator.html`)
- **模拟行情**：基于正态分布的随机价格生成，含标的和2x/3x杠杆ETF
- **模拟交易**：买入/卖出、持仓管理、浮动盈亏计算
- **业绩走势图**：Canvas绘制的净值曲线，支持多时段切换
- **交易记录**：完整的买卖历史
- **机制教育**：每日重置、波动损耗的图文说明

### ⚙️ 管理后台 (`exchange_admin.html`)
从交易所/配资方的视角管理杠杆ETF业务：

- **📊 总览**：AUM、配资敞口、杠杆率、利息收入
- **📦 产品**：各杠杆ETF的资金池详情、使用率、利率设置
- **👥 客户**：多客户持仓监控、保证金比例、风险评级
- **⚠️ 风控**：保证金充足率、集中度风险、压力测试、风控警报
- **📖 原理**：配资运作机制、每日重置实质、风控策略详解

## 杠杆ETF核心机制

```
客户自有资金 + 交易所配资金额 → 买入标的资产 → 每日重平衡
```

- **每日重置**：每个交易日结束时重新调整杠杆比例
- **波动损耗**：震荡市中杠杆ETF因重置产生的额外损耗
- **保证金制度**：初始保证金100%，维持保证金70%，强平线30%

## 使用方式

直接用浏览器打开HTML文件即可，无需安装任何依赖。

- `leveraged_etf_simulator.html` — 交易端
- `exchange_admin.html` — 管理后台

## 技术栈

纯前端 HTML + CSS + JavaScript，无任何外部依赖。
