# 杠杆ETF模拟交易平台 ⚡

一个基于中国A股标的（茅台、兆易创新、宁德时代）的杠杆产品模拟交易系统。

**身份定位：** 你是一家交易所，同时也是产品发行方。提供两种杠杆交易产品——同一份资金，两种包装。

## 版本说明

### v2.0（推荐）— 双产品线

| 页面 | 说明 | 链接 |
|------|------|------|
| `leveraged_etf_simulator_v2.html` | **前台交易端**：模式切换（杠杆ETF / 配资交易），两种交易体验 | 📈 |
| `exchange_admin_v2.html` | **后台管理端**：统一风险概览 + ETF/配资双业务线管理 | 🏦 |
| `models_analysis.html` | **模式对比分析**：杠杆ETF vs 配资的完整差异梳理 | 📋 |

### 两种产品

| | 产品A：杠杆ETF（收益互换） | 产品B：直接配资（借贷） |
|---|---|---|
| **客户体验** | 像买股票一样买卖份额 | 像期货一样管保证金 |
| **资金占用** | 按市价支付份额对价 | 存入保证金 |
| **强平机制** | 无 | 跌破30%强平 |
| **交易所收入** | 管理费（0.8%年化） | 利息（8%-12%年化） |
| **底层风险** | 交易所通过收益互换提供杠杆 | 交易所直接借贷 |

## 使用方式

直接用浏览器打开HTML文件即可，无需安装任何依赖。

## 技术栈

纯前端 HTML + CSS + JavaScript，无任何外部依赖。
